# Arbrep - Tienda de Juguetes para Adultos

Sitio web estático para tienda de juguetes para adultos con integración de WhatsApp para compras.

## Descripción

Este es un sitio web estático desarrollado para Arbrep Representaciones Comerciales, S.L., una tienda especializada en juguetes para adultos. El sitio incluye:

- Página principal con productos destacados
- Catálogo de productos
- Página de contacto
- Política de privacidad y términos de uso
- Integración con WhatsApp para compras
- Verificación de edad (solo mayores de 18 años)

## Estructura del Proyecto

```
├── index.html              # Página principal
├── productos.html          # Catálogo de productos
├── contacto.html           # Página de contacto

├── terminos.html           # Términos y condiciones
├── assets/

│   │   └── styles.css      # Estilos del sitio
│   ├── js/
│   │   ├── config.js       # Configuración
│   │   └── app.js          # Lógica de la aplicación
│   ├── img/                # Imágenes y placeholders
│   └── docs/
│       └── README-licencias.txt
└── README.md               # Este archivo
```

## Características

- **Responsive Design**: Adaptable a dispositivos móviles y desktop
- **Verificación de Edad**: Sistema de confirmación para mayores de 18 años
- **Integración WhatsApp**: Botones de compra que abren WhatsApp con información del producto
- **SEO Optimizado**: Meta tags y estructura semántica
- **Privacidad**: Configurado para no indexar en motores de búsqueda

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- WhatsApp Business API

## Uso

1. Abre `index.html` en un navegador web
2. Confirma que eres mayor de 18 años
3. Navega por los productos
4. Usa los botones de WhatsApp para contactar sobre compras

## Notas Importantes

- Este sitio está destinado exclusivamente para mayores de 18 años
- Las compras se realizan únicamente a través de WhatsApp
- El sitio incluye políticas de privacidad y términos de uso
- Los empaques son discretos para proteger la privacidad del cliente
